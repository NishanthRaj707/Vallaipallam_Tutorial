Tutorial 19: Install & Run Vallaipallam

📥 How to Download and Run Vallaipallam

In this tutorial, we will learn how to install the Vallaipallam Tamil Programming Language on your PC using pip, the Python Package Installer. We'll also learn how to run your first Tamil program from your terminal.

🔧 Step 1: Prerequisites

  • ✅ Python 3.7 or above installed on your system
  • ✅ Internet connection

📌 If you don't have Python installed, download it from python.org

📦 Step 2: Install Using pip

Open your command prompt or terminal and type:

pip install vallaipallam

This will download and install the Vallaipallam language interpreter package globally.

📁 Step 3: Folder Structure

You can create a folder anywhere (Desktop, Documents, etc) for your Vallaipallam programs.

Recommended: Create a folder called vallaipallam-projects.

📜 Step 4: Write Your First Code

Inside your project folder, create a new file with the extension .vp (e.g. firstcode.vp).

வெளியிடு("வணக்கம் உலகம்")

This is a simple print statement in Vallaipallam.

🚀 Step 5: Run the Program

Navigate to the folder using terminal:

cd path/to/vallaipallam-projects

Then run the program with:

vallaipallam firstcode.jnr

You will see the output:

வணக்கம் உலகம்

🔁 Notes

  • 🧠 You can update the language anytime using pip install --upgrade vallaipallam
  • 📝 Make sure each program ends with .vp extension
  • 📂 Keep all your codes in a dedicated folder for easy access
