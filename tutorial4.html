📦 Lesson 4: Variables

Variables are used to store data in a program. In Vallaipallam, you must use the keyword உருவாக்கு to declare a variable. Without it, the variable will not be recognized.

🛠️ Syntax:

உருவாக்கு கு = 10

Once declared, you can use the variable in functions or operations.

🔸 Example Usage:

உருவாக்கு x = 5
உருவாக்கு y = 3
வெளியிடு(x)
வெளியிடு(x + y)
வெளியிடு(x * 2)

🧾 Output

5.0
8.0
10.0

✅ Tip: Always start variable declarations with உருவாக்கு. You can perform all arithmetic operations using variables just like numbers.
