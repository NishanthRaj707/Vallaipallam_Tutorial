🔁 Lesson 11: While Loop

A while loop repeats a block of code as long as the condition is மெய் (true). This is useful when you don't know how many times a loop should run.

📘 Syntax

உருவாக்கு x = 1
அதுவரை x <= 5 {
  வெளியிடு(x)
  x = x + 1
}

🔍 Explanation

- அதுவரை means while
- The loop continues running until the condition becomes false
- Don’t forget to update the variable inside the loop, or it may become an infinite loop!

🔁 Example with Condition

உருவாக்கு i = 3
அதுவரை i > 0 {
  வெளியிடு(i)
  i = i - 1
}
3
2
1

🧾 Use Case

While loops are useful in situations like waiting for user input, retrying a task, or counting down — like we just did!
